Principais formatações!

Negrito / Destaque

Esta frase está em negrito usando uma tag não semantica:<b>!

Esta frase está em destaque com uma tag semantica:<strong>!

Itálico / Enfase

Nesta frase temos um termo em itálico usando a tag I (não semantico).

Nesta frase temos um termo em enfase usando a tag em.

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo com as tags big (não semantica) e small.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto Sublinhado / Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado com a tag U (não semântica)

Texto sobrescrito

Para inserir caracteres especiais sobrescrito, ex: x2+3

Texto subscrito

Para inserir caracteres especiais subscrito, ex:H2O
